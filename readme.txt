An extension of WooCommerce that integrate fastcheck PersonDoc in your Woocomerce login page.

== Description ==
Allow all world users to sign in/up fast and secure in your web via their Persondoc accounts

**Features provided with this plugin:**

* Completely integrate with WooCommerce plugin
* Work on myaccount page only


== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser.

1. Log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
1. In the search field type "PersonDoc" and click Search Plugins.
1. Once you’ve found it, you can install it by simply clicking Install Now button.

= Manual Installation =

1. Download the PersonDoc for WooCommerce plugin to your desktop.
1. Extract the plugin folder to your desktop.
1. Read through the "readme" file thoroughly to ensure you follow the installation instructions.
1. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
1. Go to Plugins screen and find the "PersonDoc" in the list.
1. Click Activate to activate it.

= Config plugin =

Even this plugin has been installed and activated on your site.

1. Log in to your WordPress dashboard, navigate to the PersonDoc menu and click.
2. Insert your API KEY and save.
